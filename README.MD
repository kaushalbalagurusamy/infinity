# GarmentSeg DevContainer (Claude Code, no API key)

Minimal Dev Container to use **Claude Code (Max subscription)** inside the container.

## Use
1. In VS Code / Claude Code: **Command Palette ’ Dev Containers: Reopen in Container**.
2. Verify:
   ```sh
   claude-code --version
   claude-code --help
   ```
3. Host session mounted from `~/.claude` ’ `/home/node/.claude`.
